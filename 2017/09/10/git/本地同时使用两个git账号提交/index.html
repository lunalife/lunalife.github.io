解析步骤

  1. 当push变更时，首先拿到当前项目的remote url如git@github.com:xxx/xxx.git
  2. 然后在~/.ssh/config中寻找对应的真实的url替换成真实地址git@first.github.com:xxx/xxx.git
  3. 拿到对应的identiyFile对应的ssh文件中的key提交，完成。
