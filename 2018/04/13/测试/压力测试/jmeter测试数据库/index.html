创建数据库测试计划

1 配置MySQL database driver

这个使用到了MySQL database driver，配置如下：

下载：https://www.mysql.com/products/connector/选择JDBC Driver for MySQL (Connector/J)后面的download，然后在新的页面选择jar格式下载

下载完后将文件中的mysql-connector-java-5.1.38-bin.jar文件复制到JMeter目录下的lib目录中

建立一个测试计划测试数据库服务器。

您将创建50个用户2 SQL请求发送到数据库服务器。重复该些测试100次。

总的请求数= 50 x 2 x 100 = 10’000 JDBC requests

构建这个测试计划，你需要用到下面的elements: Thread Group, JDBC Request, Summary Report.

下面的例子使用了MySQL数据库驱动程序。使用这个driver,他包含.jar文件 (例如:mysql-connector-java-X.X.X-bin.jar) 必须复制到JMeter ./lib路劲。 JMeter’s Classpath详细说明。

5 遇到的错误

错误1 ：No suitable driver found for jdbc:mysql://localhost：3306/test1

解决：没有安装驱动，参照步骤1安装

错误2 ：Could not return Connection com.mysql.jdbc.exceptions.jdbc4.CommunicationsException: Communications link failure

解决：连不上，配置出错了，返回查看发现数据库URL中的冒号写成中文输入法的冒号了
