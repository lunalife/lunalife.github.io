总结现在想到的完美组合

  1. UI自动化，只做类似表单提交有很多种组合的情况
  2. API自动化，尽量全部做完
  3. 业务常用重复劳动 vue ele flask mysql -> pyinstaller做成小工具即可
  4. 无线相关的也集成到小工具中

以上的百分比规划依赖公司测试任务量和测试人员配比情况
